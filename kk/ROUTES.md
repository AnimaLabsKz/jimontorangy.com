# Local Routes

- `/` -> home
- `/walk_in_jimon/` -> JIMON туралы
- `/we_brands/` -> Брендтеріміз
- `/product_category/` -> Жоспарлау және құрылым
- `/product/` -> Өнімдер
- `/post_list/` -> Жаңалықтар орталығы
- `/post_list?cid=1` -> redirects to `/post_list-cid-1/` (Компания жаңалықтары)
- `/post_list?cid=4` -> redirects to `/post_list-cid-4/` (Денсаулық тақырыптары)
- `/post_list?cid=5` -> redirects to `/post_list-cid-5/` (Жаңа медиа орталығы)
- `/post_detail?id=119` -> redirects to `/post_detail-119/` (119-мақала)
- `/post_detail?id=120` -> redirects to `/post_detail-120/` (120-мақала)
- `/vip_center/` -> Мүшелер аймағы
- `/contact_us/` -> Байланыс
- `/law/` -> Құқықтық мәлімдеме
