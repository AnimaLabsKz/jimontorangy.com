# Local Routes

- `/` -> home
- `/walk_in_jimon/` -> О компании JIMON
- `/we_brands/` -> Наши бренды
- `/product_category/` -> Планирование и структура
- `/product/` -> Продукты
- `/post_list/` -> Новостной центр
- `/post_list?cid=1` -> redirects to `/post_list-cid-1/` (Новости компании)
- `/post_list?cid=4` -> redirects to `/post_list-cid-4/` (Темы о здоровье)
- `/post_list?cid=5` -> redirects to `/post_list-cid-5/` (Новый медиацентр)
- `/post_detail?id=119` -> redirects to `/post_detail-119/` (Статья 119)
- `/post_detail?id=120` -> redirects to `/post_detail-120/` (Статья 120)
- `/vip_center/` -> Раздел для участников
- `/contact_us/` -> Контакты
- `/law/` -> Правовое заявление
